# Fahh - VS Code Extension

A simple VS Code extension project. Follow these steps to get it running:

### 1. Setup
*   Open the folder in **VS Code**.
*   Open the terminal inside VS Code (`Ctrl + ~`).
*   Run: `npm install`

### 2. How to Run
1.  Press **`F5`** on your keyboard. 
    *   *This opens a new window called the "[Extension Development Host]".*
2.  In that new window, press **`Ctrl + Shift + P`** (or `Cmd + Shift + P` on Mac).
3.  Type **"Hello World"** (or the command defined in the extension) and press **Enter**.

### 3. Making Changes
*   Modify the code in `src/extension.ts`.
*   Go back to the [Extension Development Host] window and press **`Ctrl + R`** to reload and see your changes.
